# btrade #
머신러닝 기반 자동 주식거래 시스템

### src/crawler ###
머신러닝을 위한 학습데이터 크롤러

### src/predictor ###
크롤링된 데이터를 기반으로 학습, 다음날의 주가를 예측하는 머신러닝 모델

### src/trader ###
예측된 데이터를 기반으로 실제 주식거래를 실행하는 클라이언트
